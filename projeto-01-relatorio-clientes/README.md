# Projeto 01 – Relatório Básico de Clientes (ABAP)

##  Objetivo
Criar um relatório simples em ABAP para leitura de dados da tabela de clientes do SAP.

##  Tecnologias Utilizadas
•⁠  ⁠ABAP
•⁠  ⁠Open SQL
•⁠  ⁠Internal Tables

##  Funcionalidades
•⁠  ⁠Leitura da tabela KNA1
•⁠  ⁠Exibição do código, nome, país e cidade do cliente
•⁠  ⁠Relatório em lista clássica

##  Conceitos Praticados
•⁠  ⁠Estrutura TYPES
•⁠  ⁠SELECT INTO TABLE
•⁠  ⁠LOOP AT
•⁠  ⁠WRITE

##  Evidências
Adicionar prints do código na SE38 e do resultado do relatório.

##  Observações
Projeto desenvolvido para aprendizado inicial em ABAP, seguindo boas práticas básicas.
